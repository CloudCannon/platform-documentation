The CloudCannon CLI manages your organizations, sites, builds, files, and form submissions from the terminal, and can generate CloudCannon configuration files for your site. It authenticates as your user and signs requests for you, so it's the quickest way to work with the [CloudCannon API](/documentation/developer-reference/api/) from a terminal or CI environment.

## Installation

Install it globally and log in:

<comp.CodeBlock language="bash" source="Terminal">
```bash
npm install --global @cloudcannon/cli
cloudcannon login
```
</comp.CodeBlock>

`cloudcannon login` creates a user access key, scoped to your individual user, which the CLI uses to act on your behalf. You can manage or revoke access keys from *Account Settings → Access Keys*.

## Full documentation

See the [CloudCannon CLI on GitHub](https://github.com/CloudCannon/cli) for the full command reference.
