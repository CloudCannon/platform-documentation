The CloudCannon SDK is a typed client for the [CloudCannon API](/documentation/developer-reference/api/), with a sub-client for each resource and built-in pagination, sorting, and filtering. It handles authentication and request signing for you, so reach for it before calling the HTTP endpoints directly.

## Installation

<comp.CodeBlock language="bash" source="Terminal">
```bash
npm install @cloudcannon/sdk
```
</comp.CodeBlock>

## Usage

Create a client with your API key, then call a resource:

<comp.CodeBlock language="typescript" source="example.ts">
```typescript
import CloudCannonClient from '@cloudcannon/sdk';

const client = new CloudCannonClient({ key: '<your-api-key>' });

const { items } = await client.orgs();
```
</comp.CodeBlock>

API keys are scoped to an organization. Generate one from your organization's *API Keys* settings. For more detail on credentials and request signing, see the [API reference](/documentation/developer-reference/api/#authentication).

## Full documentation

See the [CloudCannon SDK on GitHub](https://github.com/CloudCannon/sdk) for the full documentation and the complete list of resources.
