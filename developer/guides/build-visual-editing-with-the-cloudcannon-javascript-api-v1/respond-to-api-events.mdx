In this step, we'll subscribe to v1 events so your custom UI updates automatically.

1. Register `change` listeners on the API or file objects.
2. Update your rendering function when events fire.
3. Remove listeners when cleaning up.

<comp.CodeBlock language="javascript" source="src/event-sync.js">
```
import { withCloudCannonApi } from './cloudcannon-api';

withCloudCannonApi((api) => {
  const refresh = async () => {
    const file = api.currentFile();
    const data = await file.data.get();
    renderPreview(data);
  };

  api.addEventListener('change', refresh);
  api.addEventListener('delete', refresh);

  window.addEventListener('beforeunload', () => {
    api.removeEventListener('change', refresh);
    api.removeEventListener('delete', refresh);
  });
});
```
</comp.CodeBlock>
