In this step, we'll initialize JavaScript API v1 and keep initialization logic reusable.

1. Create a helper that checks `window.CloudCannonAPI`.
2. Fall back to the `cloudcannon:load` event when needed.
3. Call your setup callback with `useVersion('v1')`.

<comp.CodeBlock language="javascript" source="src/cloudcannon-api.js">
```
export function withCloudCannonApi(callback) {
  const setup = (router) => {
    const api = router?.useVersion('v1');
    if (api) callback(api);
  };

  if (window.CloudCannonAPI) {
    setup(window.CloudCannonAPI);
    return;
  }

  document.addEventListener(
    'cloudcannon:load',
    (event) => setup(event.detail?.CloudCannonAPI),
    { once: true }
  );
}
```
</comp.CodeBlock>
