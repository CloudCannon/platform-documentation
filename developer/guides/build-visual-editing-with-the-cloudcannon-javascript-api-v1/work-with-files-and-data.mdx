In this step, we'll read and update both structured data and body content.

1. Access the active file with `currentFile()`.
2. Update a structured field with `file.data.set()`.
3. Append content with `file.content.get()` and `file.content.set()`.

<comp.CodeBlock language="javascript" source="src/file-updates.js">
```
import { withCloudCannonApi } from './cloudcannon-api';

withCloudCannonApi(async (api) => {
  const file = api.currentFile();

  await file.data.set({
    slug: 'title',
    value: 'Updated from JavaScript API v1'
  });

  const content = await file.content.get();
  await file.content.set(`${content}\n\nUpdated in this guide.`);
});
```
</comp.CodeBlock>
