This guide covers how to add <comp.GlossaryTerm term="/user/glossary/i/internationalization.yml">internationalization (i18n)</comp.GlossaryTerm> and multilingual support to your static site on CloudCannon using <comp.GlossaryTerm term="/user/glossary/r/rosey.yml">Rosey</comp.GlossaryTerm>. You'll learn how Rosey works and how to install and configure it to build locale-specific versions of your *Site*.

Before you begin, this guide assumes you already have a static *Site* connected to CloudCannon.

<comp.Notice info_type="info">
  This is a fresh setup guide. If you're migrating from CloudCannon's deprecated built-in i18n, see the [Rosey migration guide](/documentation/developer-guides/migrate-from-legacy-internationalization-to-rosey/) instead.
</comp.Notice>
