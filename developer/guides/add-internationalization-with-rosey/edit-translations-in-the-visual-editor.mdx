With the *Rosey CloudCannon Connector* installed and configured, your team can switch between locales, edit translations inline, and identify stale translations without leaving the *Visual Editor*.

## Use the locale switcher in the Visual Editor

When your *Site* loads in the *Visual Editor* with the Connector installed, a floating locale switcher appears in the corner of the preview.

<comp.DocShot docshot_key="Visual-Editor-Rosey-Locale-Switcher" alt="The Visual Editor preview with a floating locale switcher button visible in the corner." title="The locale switcher in the Visual Editor." type="ui-snippet" />

Click the locale switcher to open a list of your available locales. Selecting a locale reloads the preview in that language, showing the translated content inline.

## Edit translations inline

Click any element with a `data-rosey` attribute to open its inline translation input. The input shows the current source text and lets you enter or update the translation for the active locale.

<comp.DocShot docshot_key="Visual-Editor-Rosey-Inline-Edit" alt="A translatable element selected in the Visual Editor, with an inline input showing the source text and a field to enter a translation." title="Editing a translation inline in the Visual Editor." type="ui-snippet" />

When you save, the Connector writes the updated value back to the locale JSON file.

## Resolve stale translations

When source content changes after a translation has been saved, the Connector highlights the affected elements with amber dashed borders. A badge on the locale switcher shows how many stale translations exist on the current page.

<comp.DocShot docshot_key="Visual-Editor-Rosey-Stale-Translations" alt="The Visual Editor showing translatable elements with amber dashed borders, and a badge on the locale switcher indicating the number of stale translations." title="Stale translations highlighted in the Visual Editor." type="ui-snippet" />

To resolve a stale translation, click the highlighted element to open its inline translation input, review the updated source text, and enter a new translation. The amber highlight clears once the translation matches the current source text.
