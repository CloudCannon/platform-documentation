Welcome to this guide on building custom visual editor interfaces with JavaScript API v1.

In this guide, we'll initialize the API, update file data and content, work with collections, subscribe to change events, and open custom data panels for editors.

Before we begin, this guide assumes your pages already render in CloudCannon's *Visual Editor*.
