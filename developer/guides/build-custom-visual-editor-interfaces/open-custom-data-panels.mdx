In this step, we'll open a custom data panel and apply updates to your interface.

1. Call `createCustomDataPanel()` with a title, data object, and config.
2. Handle changes in `onChange`.
3. Apply updated values to your custom preview.

<comp.CodeBlock language="javascript" source="src/custom-panel.js">
```
import { withCloudCannonApi } from './cloudcannon-api';

withCloudCannonApi(async (api) => {
  await api.createCustomDataPanel({
    title: 'Hero settings',
    data: {
      heading: 'Welcome',
      showCta: true
    },
    config: {
      _inputs: {
        heading: { type: 'text' },
        showCta: { type: 'boolean' }
      }
    },
    onChange: (updatedData) => {
      renderHeroPreview(updatedData);
    }
  });
});
```
</comp.CodeBlock>
