To add a [Client Sharing](/documentation/articles/what-is-client-sharing/) password to your Site:

1. Navigate to the *Client Sharing* page under *Site settings*.
2. Under the *Password* tab, set the password to log in to the Site. For security reasons, the password must be at least six characters long
3. Click the *Set password and turn on* button to enable Client Sharing for this Site.

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/sharing/client-sharing/2024-05/CloudCannon-Documentation-Enable-Client-Sharing-Page.png" alt="A screenshot of the Client Sharing page shows a field for setting the Client password and a button to enable Client Sharing." title="The Client Sharing page" type="screenshot"/>

Clients can now log in to your Site on the Client Login Page using the password you set. The Client Login Page will be the Site domain with the suffix “/update” unless you have [configured the URL to use a different subpath](/documentation/articles/configure-the-client-login-page-and-dashboard/).

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/sharing/client-sharing/2024-05/CloudCannon-Documentation-Update-Client-Sharing-Password.png" alt="A screenshot of the Password tab on the Client Sharing page shows a field to reset the Client password and a button to disable Client Sharing." title="The Client Sharing page" type="screenshot"/>

To open the Client Login Page for your Site, click the *Open Client Login Page* button.

To turn off Client Sharing for a Site, click the *Turn Off* button at the top of the *Client Sharing* page.
