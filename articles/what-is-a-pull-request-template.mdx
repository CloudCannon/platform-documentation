A Pull Request template allows you to define the default values for the title and body of your Pull Requests.

&nbsp;

&nbsp;

You can also configure any [inputs](/documentation/articles/what-are-inputs/ "What are inputs?") used by your Pull Request. If you have multiple templates on your *Site*, you can choose which template you want to use when creating a Pull Request.
