The Content Editor is a rich text editor for Markdown files.

Editors use the Content Editor to update Markdown in a clean and intuitive interface. This is particularly useful for blog posts and other content-heavy web pages.

<comp.Notice info_type="info">
  Change the [toolbar controls for the Content Editor](/documentation/articles/defining-editable-regions-in-your-html/#options) with the `_editables.content` option.
</comp.Notice>
