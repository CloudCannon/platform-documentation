The Content Editor is a rich text editor for Markdown files.

Editors use the Content Editor to update Markdown in a clean and intuitive interface. This is particularly useful for blog posts and other content-heavy web pages.

<comp.DocsImage path="ye_olde_images/editing-editors-content-editor.png" alt="Screenshot of Content Editor with sidebar closed" type="screenshot" />

<comp.Notice info_type="info">
  Change the [toolbar controls for the Content Editor](/documentation/articles/defining-editable-regions-in-your-html/#options) with the `_editables.content` option.

  If you only want to edit the front matter of a Markdown file, you can [hide the Content Editor](/documentation/articles/choosing-the-editors-for-each-file/#hiding-the-content-area) with the `_enabled_editors` option, 
</comp.Notice>
