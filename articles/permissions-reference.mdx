You can control what your team can access using <a href="/documentation/articles/default-permission-groups/" target="_blank" rel="noopener">permission groups</a>. For each permission group, you can set the permissions for every member of that group.

In CloudCannon, [default](/documentation/articles/default-permission-groups/) and [custom permission groups](/documentation/articles/create-a-permission-group/) are available depending on your [pricing plan](/pricing/). You cannot configure the permissions for default permission groups.

This article covers all permissions in CloudCannon. To learn more about permission groups, read our documentation on [creating a permission group](/documentation/articles/create-a-permission-group/).
