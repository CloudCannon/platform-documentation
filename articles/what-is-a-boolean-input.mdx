A Boolean input is an editing interface for true or false values. You can use this input to edit structured data in data files or front matter using the [Data Editor](/documentation/articles/the-data-editor/), or data panels and the sidebar in the [Content Editor](/documentation/articles/the-content-editor/) or [Visual Editor](/documentation/articles/the-visual-editor/).

There are two types of Boolean input:

* Checkbox
* Switch

## Boolean input types

### Checkbox

The Checkbox Boolean input provides an editing interface for true or false values. True values have a checked box, while false values have an unchecked box.

You can change the value of a Checkbox input by clicking on the checkbox.

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/editing/inputs/boolean/2024-07/CloudCannon-Documentation-Checkbox-Boolean-Input.png" alt="A screenshot of two Checkbox Boolean inputs in the Data Editor, showing one true and one false value." title="The Checkbox Boolean Input" type="ui-snippet"/>

### Switch

The Switch Boolean input provides an editing interface for true or false values. True values show the switch to the right with the text "ON", while false values show the switch to the left with the text "OFF".

You can change the value of a Switch input by clicking on the switch.

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/editing/inputs/boolean/2024-07/CloudCannon-Documentation-Switch-Boolean-Input.png" alt="A screenshot of two Switch Boolean inputs in the Data Editor, showing one true and one false value." title="The Switch Boolean Input" type="ui-snippet"/>

## Boolean input appearance and behavior

Boolean inputs are simple. You cannot configure additional input behavior besides the general configuration options.

For more information, please read our developer documentation on [configuring a Boolean input](/documentation/articles/configure-a-boolean-input/) for configuration options, valid input values, and unconfigured input behavior.

## Misconfigured Boolean inputs

You will see an orange warning box if your Boolean input is misconfigured.

<comp.DocShot docshot_key="" alt="" title=""/>

Boolean inputs are misconfigured if they contain any value other than `true` or `false`. For more information on how to fix a misconfigured Boolean input, please read our developer documentation on [configuring a Boolean input](/documentation/articles/configure-a-boolean-input/).
