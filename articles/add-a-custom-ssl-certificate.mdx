<comp.Notice info_type="important">
  Managing your SSL certificate on CloudCannon requires you to host your Site through CloudCannon. If you are hosting your Site through an external service, please review their documentation on SSL certification.
</comp.Notice>


Only sites with [Custom Domains](/documentation/articles/what-is-a-custom-domain/) can add custom SSL certificates.

To add a new custom SSL certificate:

1. Navigate to the *Custom Domain* page under *Site Settings*, and click on the *SSL* tab.
2. Click on the *Custom SSL certificate* dropdown and select the *New custom SSL certificate* option.
3. In the *Add SSL certificate* modal, enter the details for Name, Expires, Public Key, Private Key, Certificate Chain.
4. Click the *Add SSL certificate* button.
5. Click the *Update SSL* button.

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/hosting/ssl/2024-08/CloudCannon-Documentation-Add-SSL-Certificate-Modal.png" alt="A screenshot of the Add SSL Certificate modal shows fields for Name, Expires, Public Key, Private Key, Certificate Chain." title="The Add SSL Certificate modal" type="screenshot"/>

Alternatively, you can leave the *Custom SSL certificate* dropdown blank and CloudCannon will autogenerate an SSL certificate for you.

<comp.Notice info_type="info">
  All certificates must use PEM encoding. The placeholder text indicates the correct start and end headers.
</comp.Notice>
