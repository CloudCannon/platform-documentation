This release added the Update from Publish Branch feature and unlimited trial periods for Partner Client Organizations. It also addressed issues affecting the&nbsp;*Publishing*&nbsp;tab on the&nbsp;*Project*&nbsp;page, commit links on the&nbsp;*Activity*&nbsp;page, and merge requests from GitLab.

### Features:

* Added the&nbsp;<a target="_blank" href="https://devout-geyser.cloudvent.net/documentation/articles/connect-a-publish-branch/#update-from-publish-branch">Update from Publish Branch</a>&nbsp;feature.&nbsp;
  * CloudCannon will detect when your Publish Branch has been updated. When updates are available, a red notification icon will appear on the&nbsp;*Publish*&nbsp;button in your&nbsp;*Site navigation*.
  * You can now pull these updates to your branched site through the&nbsp;*Update this site*&nbsp;button on the&nbsp;*Publishing*&nbsp;page.
* Partners now have access to unlimited trial periods for new clients.

### Fixes:

* Fixed an issue where images and links in markdown were escaped if the filepath included a space.
* Fixed an issue where the&nbsp;*Publishing*&nbsp;tab on the&nbsp;*Project*&nbsp;page was not visible to users with Editor-level permissions.
* Fixed an issue where the&nbsp;*Publishing*&nbsp;tab diagram did not correctly display site relationships.
* Fixed an issue where the&nbsp;*Redirects*&nbsp;tab on the&nbsp;*Domain Settings*&nbsp;page did not display site icons correctly.
* Fixed an issue where the&nbsp;*Activity*&nbsp;page failed to link to commits.
* Fixed an XSS vulnerability.
* Improved visibility of pending or blocked merge requests from GitLab and added more detailed error messages.
* Updated dependencies to patch security vulnerabilities.
