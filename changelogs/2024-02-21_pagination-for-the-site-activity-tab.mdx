This release

### Features & Improvements:

* Added pagination for the *Site Activity* tab.
* Minor copy improvements in the app.
* Users can now sync files while an editing session is open.

### Fixes:

* This release fixed an issue preventing Standard Plan users from adding multiple Site Mounting.
* This release fixed an issue where the Cloudflare icon was not visible on a white background.
* This release fixed an issue causing domain releases to result in a 404 error.
* This release fixed an issue where CloudCannon would not redirect users away from the *Domain* page after they released a domain.
* This release fixed an issue preventing users from setting a DAM as the main source.
* This release fixed an issue causing the *Sync files* button on the *Site dashboard* to fail in the event of an error.
