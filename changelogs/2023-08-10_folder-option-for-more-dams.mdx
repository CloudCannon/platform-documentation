title: Folder option for more DAMs
type: minor

The "Extra Prefix" option can now be set in the site-level settings for Cloudflare R2, DigitalOcean Spaces, and Google Cloud Storage DAMs. This options restricts all uploads and browsing to a specified folder.

**Features:**

* "Extra prefix" option is now supported for Cloudflare R2, DigitalOcean Spaces, and Google Cloud Storage DAMs.

**Fixes:**

* Fixed a bug related to snippets inside rich text inputs.
