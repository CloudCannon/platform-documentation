This release updated our legacy Eleventy plugin.

It also addressed several issues affecting discarding unsaved changes, Datetime inputs, and Snippets in the Visual Editor.

## Features & Improvements

* Updated dependencies to patch security vulnerabilities. for `eleventy-plugin-cloudcannon`.
* Added terminal colors to non-Unified Eleventy Sites.
* Updated dependencies to patch security vulnerabilities.

## Fixes

* Fixed an issue where your could not discard unsaved changes from the *Context menu* in an editing interface.
* Fixed an issue where, in some cases, CloudCannon would format the value of Datetime inputs as a string.
* Fixed an issue where, in some cases, CloudCannon did not display Snippets inside code blocks in a Markdown file correctly in the Visual Editor.
