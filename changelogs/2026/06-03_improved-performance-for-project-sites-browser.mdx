This release improved performance when listing Sites in the Project Sites Browser and updated the default sorting method across several areas of the app to last-synced.

## Features & Improvements

* Improved performance for the *Project Sites Browser*, particularly for *Organizations* with custom *Permission Groups* and many *Exceptions*.
* CloudCannon now defaults to sorting the *Project Browser*, *Site Browser*, *Branches*, and *Pull Requests* by last synced first.
