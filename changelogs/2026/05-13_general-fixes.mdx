This release addressed several issues, including those affecting the *Reset using Git repository* option, *Select* fields, and the *Choose a collection* modal.

## Fixes

* Fixed an issue where the *Reset using Git repository* option did not perform a forced reconnect.
* Fixed an issue where you could not navigate *Select* fields using the keyboard.
* Fixed an issue where *Select* fields had no height in Safari.
* Fixed an issue where collection groups with no heading were not shown in the *Choose a collection* modal.
* Updated dependencies to patch security vulnerabilities.
