This release made it easier to move between your *Organizations*, manage invitations, and create new *Organizations* from the *Account Menu*.

It also reworked the *Account Menu* so it offers the same options everywhere in CloudCannon, added a landing page for new accounts created from an invitation, and fixed an issue where your *Organizations* list did not update after accepting an invitation.

## Features & Improvements

* Added a direct sign-in for single-*Organization* accounts. If you belong to exactly one *Organization*, CloudCannon now takes you straight into it when you open the app, instead of showing the *Organizations Browser* first. Because you no longer pass through the *Organizations Browser*, the *Account Menu* now provides everything you previously did there — responding to invitations and creating a new *Organization* — so you keep full access to those actions.
* Moved responding to invitations into the *Account Menu*. A *Pending invitations* option, with a badge on your *Avatar*, now appears whenever you have invitations waiting, and opens a modal where you can *Accept* or *Decline* each one without leaving your current page. You previously did this from the *Organizations Browser*.
* Moved creating a new *Organization* into the *Account Menu*. A *Create new organization* option now opens a modal so you can create an *Organization* directly. You previously did this from the *Organizations Browser*.
* Reworked the *Account Menu* so it offers the same options whether you are working inside an *Organization* or viewing the *Organizations Browser*.
* Added a better landing page for new accounts created from an email invitation, showing your pending invitations so you can *Accept* one or create your own *Organization*.

## Fixes

* Fixed an issue where accepting an invitation did not update your *Organizations* list or the rest of the app until you refreshed the page.
