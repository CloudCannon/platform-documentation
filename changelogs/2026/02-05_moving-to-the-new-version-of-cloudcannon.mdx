The [new version](/documentation/changelog/2025/11/6/updated-editable-regions-configuration-mode-and-a-ui-overhaul/) of the CloudCannon app is now the default for everyone. Thank you to everyone who has provided feedback on our beta so far. We will continue to support the Legacy version of CloudCannon until February 12th 2026. If you have any further feedback or questions, please don't hesitate to use the [CloudCannon Community](https://community.cloudcannon.com/t/cloudcannon-new-version-launch-february-5th-2026/397) or contact our friendly [support team](https://actual-badger.cloudvent.net/support/).

This release also updated the CloudCannon Documentation website.

## Features & Improvements

<comp.DocsImage path="https://cc-dam.imgix.net/documentation/images/change-log/2026-02/CloudCannon-Changelog-Feb2026-New-Documentation-Light-Dark-Mode.png" alt="An illustration of the new documentation website showing Light and Dark Mode options." title="Light and Dark Mode on CloudCannon Documentation" type="screenshot"/>

* Updated the appearance and functionality of the CloudCannon Documentation website.
  * Split content into User and Developer sections for different audiences.
  * Organized content into Articles, Guides, Glossary, and References to support reader goals.
  * Added the Reference section to act as the source of truth for all configuration keys in CloudCannon.
  * Updated to the [newest Pagefind engine](https://github.com/Pagefind/pagefind/discussions/1009) for better search results.
  * Added Light and Dark Mode, which respects browser defaults.
  * Complete UI overhaul.
  * Read more about the redesign process in our most recent [blog](/blog/redesigning-cloudcannons-docs-with-diataxis-lume-and-pagefind/).
