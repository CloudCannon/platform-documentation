This release added pagination for sections of the *Subscription* page.

It also addressed several issues affecting add-on warning emails, stripping markup in the Content Editor, billing information for trial users, and filters in the *Domains browser*.

## Features & Improvements

* Added pagination for items in the *Monthly Subscription* section on the *Subscription* page.

## Fixes

* Fixed an issue where warning emails about monthly bandwidth and build-time usage were not sent correctly.
* Fixed an issue where CloudCannon did not strip class names from pasted text in the Content Editor.
* Fixed an issue where, in some cases, users on a CloudCannon trial would see an incorrect billing number on the *Subscription* page.
* Fixed incorrect text in the *Domains browser* filter options.
