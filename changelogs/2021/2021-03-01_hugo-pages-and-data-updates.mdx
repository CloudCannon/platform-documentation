This release adds a few extra features to the Hugo private beta as well as a few bug fixes.

**Features:**

* Pages working on Hugo sites
* Default create flows for data subfolders
* Default add actions for data files in subfolders

**Fixes:**

* Fixed geolocation detection on apex domains
* Redirect wildcard to main site uses 307 instead of a 301
* CloudCannon sitemap overriding custom sitemap
* Some developer files are hidden in Client Sharing interface
