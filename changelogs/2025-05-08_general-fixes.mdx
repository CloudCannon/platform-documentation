This release addressed several issues, including those affecting Collections, Image inputs with resize options, and file names with trailing spaces.

## Fixes

* Fixed an issue where, in some cases, your Collections would not appear in the *Site Navigation* after the initial build.
* Fixed an issue where you could not select images from a Cloudinary DAM using an Image input with resize options enabled.
* Fixed an issue where CloudCannon did not support file names with trailing spaces.
