title: Cancel build button
type: minor
---
This release adds the ability to cancel builds when they are running from the site status page. Also included are a variety of security improvements and general fixes.

**Features:**

* A new "Cancel build" button in site status
* Changed CleanCSS to esbuild CSS minification

**Fixes:**

* Removing billing prompt for partner program users
* File move activities show the correct paths
* Fixed visual editing for `.liquid` files
* Upgraded JQuery to latest
* Upgraded websocket security
* General security fixes
