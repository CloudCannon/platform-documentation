CloudCannon Legacy is no longer available, as we enter into a new private beta.

**Features:**

* CloudCannon Legacy is no longer available.
* app.cloudcannon.com now returns CSP headers and uses TLS v1.3.
* Site backups will now be automatically deleted after 30 days, except for the 2 most recent backups.

**Fixes:**

* Fixed an error that could occur when selecting an SVG for a rich text image input.
* Fixed an issue causing some builds to fail.
