This release changed the *Toggle Navigation* button for a *Return to Site* button.

It also fixed an issue affecting .ruby-version files and creating new Sites from a local folder.

### Features & Improvements:

* The *Toggle Navigation* button in the editing interface has been replaced with a *Return to Site.*

### Fixes:

* Fixed an issue preventing you from adding a .ruby-version file.
* Fixed an issue preventing you from creating a site from a local folder.
