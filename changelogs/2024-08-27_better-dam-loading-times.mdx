title: Better DAM loading times
---
This release improved loading times for DAM browsers.

It also addressed several issues, including those affecting Site loading, the *Organizations* page, the Dam browser, and Datetime inputs.

## Features & Improvements

* Improved loading times when browsing DAM assets.

## Fixes

* Fixed an issue preventing some Sites from loading.
* Fixed an issue causing the *Organizations* page to have a 500 error.
* Fixed an issue where, in some cases, removing an Azure DAM broke the DAM browser.
* Fixed an issue where, in some cases, editing a Datetime input did not update the content to match a configured timezone.
