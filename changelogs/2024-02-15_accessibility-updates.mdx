This release improved accessibility in the CloudCannon app by updating the shade of red used in the terminal and adding keyboard accessibility for various buttons.

It also addressed incorrect copy in the upstream changes tooltip, extra prefixes preventing filename clash detection when uploading to a DAM, and patched security vulnerabilities.

### Features & Improvements:

* Updated the shade of red used in the CloudCannon terminal to pass Web Content Accessibility Guidelines (WCAG) criteria at AAA level.
* Improved keyboard accessibility for various buttons in the app.

### Fixes:

* Fixed the tooltip copy regarding&nbsp;<a href="http:///documentation/articles/update-from-a-publish-branch/" target="_blank" rel="noopener">upstream changes</a>&nbsp;on the&nbsp;*Publish*&nbsp;button in your&nbsp;*Site navigation*.
* CloudCannon now respects extra prefixes when checking for clashing filenames while uploading to a DAM.
* Updated dependencies to patch security vulnerabilities.
