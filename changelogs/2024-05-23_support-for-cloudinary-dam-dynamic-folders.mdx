This release add support for Cloudinary DAMs using dynamic folder mode, updated our dropdown menus, and improved the clarity of copy in the app.

## Features & Improvements

* Added support for Cloudinary DAMs using dynamic folder mode, including the appropriate directory structure and asset names.
* Updated our dropdown menus in the app for better styling, accessibility, and placement.
* Improved the clarity of copy in the app regarding Client Sharing.
