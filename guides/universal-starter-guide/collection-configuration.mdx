    ## CloudCannon Reader

    To integrate your site’s collections into the CloudCannon app, we run the open-source NPM library&nbsp;<a target="_blank" rel="noopener" href="https://github.com/CloudCannon/reader">CloudCannon reader</a>. **You don’t** **need to install this package, as CloudCannon will run it automatically as part of your build processes.**

    The CloudCannon reader tool will scan your global configuration file to find pages, collections, and data files. The reader will write this information to the hidden file&nbsp;`_cloudcannon/info.json`. When you navigate to a collection from your sidebar, the preview of each file and other elements in the CloudCannon app interface are generated using this JSON file.
