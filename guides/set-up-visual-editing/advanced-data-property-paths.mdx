## Special data sources

The `data-prop` HTML attribute is not restricted to structured data keys. In a previous step of this guide [Editable Regions for content files](/documentation/guides/set-up-visual-editing/editable-regions-for-content-files/), we introduced the `@content` special value, which allows you to edit the Markdown content of a file. There are several other special data source values you can use to edit data in various places.

## Step 8: Advanced Patterns

### Conditional Content Blocks

For more complex layouts with different content block types:

```markdown
---
title: "Homepage"
content_blocks:
  - type: "hero"
    headline: "Welcome"
    image: "/hero.jpg"
  - type: "text"
    content: "Some text content"
  - type: "gallery"
    images: ["/img1.jpg", "/img2.jpg"]
---
```

```astro
<array-editable data-prop="content_blocks" data-id-key="type" data-component-key="type">
  {frontmatter.content_blocks?.map((block, index) => (
    <array-item key={index} data-id={block.type} data-component={`${block.type}_block`}>
        <Component {...block} />
    </array-item>
  ))}
</array-editable>
```
